# WDPAI
## ZakupyPro: Zarządzanie Listami Zakupów

Aplikacja webowa umożliwiająca użytkownikom tworzenie, edytowanie i zarządzanie listami zakupów. Użytkownicy będą mogli kopiować wcześniej utworzone listy oraz zapisywać je do późniejszego użytku. Aplikacja automatycznie obliczy średnią wartość zakupów na podstawie dotychczasowych wydatków.
